BetaXShell 内置强大的 xterm.js 终端模拟器，提供完整的 SSH 终端体验。本指南将帮助你掌握终端的各项功能。

BetaXShell 终端由以下主要部分组成：

- <Icon name="list" size="1em" /> **标签页栏**：显示和管理多个终端会话
- <Icon name="toolbar" size="1em" /> **工具栏**：快速访问常用功能
- <Icon name="desktop" size="1em" /> **终端区域**：显示和执行命令
- <Icon name="scroll" size="1em" /> **滚动条**：浏览历史输出

### 标签页管理

<Icon name="plus" size="1.2em" /> **新建标签页**

- 点击 **"+"** 按钮创建新标签页
- 快捷键：`Ctrl + T`
- 自动打开新的终端会话

<Icon name="x-circle" size="1.2em" /> **关闭标签页**

- 点击标签页的 **"×"** 按钮
- 快捷键：`Ctrl + W`
- 双击标签页也可以关闭

<Icon name="arrows-left-right" size="1.2em" /> **切换标签页**

- 点击标签页名称切换
- 快捷键：`Ctrl + Tab`（下一个）
- 快捷键：`Ctrl + Shift + Tab`（上一个）

## <Icon name="keyboard" size="1.2em" /> 快捷键大全

### 通用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + T` | 新建标签页 |
| `Ctrl + W` | 关闭当前标签页 |
| `Ctrl + Tab` | 切换到下一个标签页 |
| `Ctrl + Shift + Tab` | 切换到上一个标签页 |
| `Ctrl + F` | 在终端中查找 |
| `F11` | 全屏模式 |

### 文本操作

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Shift + C` | 复制选中文本 |
| `Ctrl + Shift + V` | 粘贴文本 |
| `Ctrl + Insert` | 复制（备用） |
| `Shift + Insert` | 粘贴（备用） |

### 字体调整

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Plus` | 增大字体 |
| `Ctrl + Minus` | 减小字体 |

| 快捷键 | 功能 |

| `Ctrl + L` | 清屏（等同于 clear 命令） |

## <Icon name="scroll" size="1.2em" /> 滚动和缓冲区

### 滚动操作

- <Icon name="arrow-up" size="1em" /> **向上滚动**：查看历史输出
- <Icon name="arrow-down" size="1em" /> **向下滚动**：返回最新输出
- <Icon name="home" size="1em" /> **滚动到顶部**：查看最早的输出
- <Icon name="end" size="1em" /> **滚动到底部**：返回当前行

### 缓冲区设置

<Icon name="database" size="1.2em" /> 调整终端缓冲区大小：

1. 进入 **设置** > **终端** > **缓冲区**
2. 设置 **滚动行数**：
   - 默认：1000 行
   - 最小：100 行
   - 最大：10000 行
3. 设置 **启用滚动备份**

<Icon name="lightbulb" size="1em" color="var(--sl-color-accent)" /> **提示**：较大的缓冲区会占用更多内存，根据需要调整。

## <Icon name="bell" size="1.2em" /> 终端铃声和通知

### 视觉铃声

<Icon name="eye" size="1em" /> 当终端发送响铃字符（`\a`）时：

- 终端窗口闪烁
- 标签页高亮显示
- 不会发出实际声音

### 通知设置

<Icon name="gear" size="1.2em" /> 配置终端通知：

1. 进入 **设置** > **终端** > **通知**
2. 启用以下选项：
   - <Icon name="bell" size="1em" /> **长时间命令完成时通知**：命令执行超过 30 秒时显示通知
   - <Icon name="bell" size="1em" /> **后台任务完成时通知**：后台任务完成时提醒
   - <Icon name="bell" size="1em" /> **终端响铃时通知**：收到响铃字符时通知

## <Icon name="text" size="1.2em" /> 文本编码

### 字符编码支持

BetaXShell 支持多种字符编码：

- <Icon name="checkmark" size="1em" /> **UTF-8**（推荐）：默认编码，支持所有 Unicode 字符
- <Icon name="checkmark" size="1em" /> **GBK**：中文编码
- <Icon name="checkmark" size="1em" /> **Big5**：繁体中文编码
- <Icon name="checkmark" size="1em" /> **ISO-8859-1**：西欧编码

### 设置编码

1. 编辑主机配置
2. 在 **高级设置** 中选择 **字符编码**
3. 保存后重新连接

## <Icon name="code" size="1.2em" /> 特殊字符支持

### Emoji 支持

<Icon name="smiley" size="1em" /> 终端完全支持 Emoji 显示：

```
echo "✅ 成功" "❌ 失败" "⚠️ 警告" "ℹ️ 信息"
```

### 控制序列

支持 ANSI 转义序列：

- **颜色**：`\033[31m红色文本\033[0m`
- **粗体**：`\033[1m粗体文本\033[0m`
- **下划线**：`\033[4m下划线文本\033[0m`
- **光标移动**：`\033[10;20H` 移动到第 10 行第 20 列

## <Icon name="bug" size="1.2em" /> 常见问题

### 终端显示乱码

**问题**：终端显示乱码或方框

**解决方案**：
1. 检查服务器字符编码设置：`echo $LANG`
2. 在主机配置中设置正确的字符编码
3. 安装必要的字体：`sudo apt install fonts-noto`

### 颜色显示不正常

**问题**：终端颜色显示不正确

**解决方案**：
1. 检查终端颜色方案设置
2. 确保应用程序支持彩色输出
3. 尝试启用 **"强制颜色"** 选项

### 终端响应慢

**问题**：终端输入有延迟

**解决方案**：
1. 检查网络连接稳定性
2. 减小终端缓冲区大小
3. 禁用不必要的终端功能

### 复制粘贴失败

**问题**：无法复制或粘贴文本

**解决方案**：
1. 确保已选中文本
2. 检查剪贴板访问权限
3. 尝试使用快捷键 `Ctrl+Shift+C/V`
