# 项目介绍

BetaXShell 是一款现代化的 SSH 客户端，为开发者、运维人员和系统管理员设计。它结合了强大的功能与优雅的用户界面，提供高效的远程服务器管理和文件传输体验。

## 核心特性

### 多协议支持

- **SSH 终端**: 完整的 SSH 协议支持，包括多种认证方式（密码、SSH 密钥）
- **SFTP 文件传输**: 集成的 SFTP 客户端，支持文件和文件夹的上传下载
- **多会话管理**: 通过标签页同时管理多个 SSH 连接

### 智能主机管理

- **分组管理**: 按项目、环境或用途组织服务器
- **快速搜索**: 即时过滤和查找主机
- **收藏功能**: 标记常用主机快速访问
- **连接历史**: 自动记录最近连接的主机

### 高级文件传输

- **文件夹同步**: 智能同步本地和远程文件夹
- **断点续传**: 网络中断后可继续传输
- **全局传输浮标**: 浮动按钮显示活跃任务数，点击查看所有传输任务
- **传输抽屉**: 统一管理所有传输任务，支持暂停/继续/重试/取消
- **实时统计**: 显示传输速度、进度、预计完成时间

### 安全优先

- **系统密钥链集成**: 使用操作系统密钥链安全存储密码
- **主机指纹验证**: SSH 主机密钥验证，防止中间人攻击
- **加密存储**: 敏感信息使用 AES-256-GCM 加密
- **密钥管理**: 支持 SSH 私钥导入和管理

### 用户体验

- **系统托盘集成**: 最小化到系统托盘，快速访问常用主机
- **快捷命令**: 预定义常用命令，一键执行
- **自动更新**: 内置自动更新机制
- **跨平台**: 支持 Windows、macOS 和 Linux

## 技术架构

BetaXShell 采用三层架构设计，充分利用各技术栈的优势：

### 前端层 (Vue 3)

负责用户界面和交互逻辑：

- **Vue 3**: 响应式 UI 框架
- **TypeScript**: 类型安全的开发体验
- **Naive UI**: 现代化的组件库
- **Pinia**: 状态管理
- **xterm.js**: 终端模拟器
- **Vite**: 快速的开发构建工具

### 中间层 (Tauri + Rust)

连接前端和后端的桥梁：

- **Tauri 2.x**: 轻量级跨平台桌面框架
- **进程管理**: 管理 Go 后端进程生命周期
- **IPC 通信**: 前后端之间的消息传递
- **系统调用**: 文件系统、对话框等系统 API
- **自动更新**: 应用程序更新机制

### 后端层 (Go)

处理核心业务逻辑：

- **SSH 协议**: 基于 `golang.org/x/crypto/ssh`
- **SFTP 协议**: 基于 `github.com/pkg/sftp`
- **WebSocket**: 实时终端输出传输
- **Gin 框架**: HTTP API 服务
- **系统密钥链**: `github.com/zalando/go-keyring`
- **SQLite**: 本地数据存储

## 工作原理

### 连接流程

1. 用户通过前端界面发起连接请求
2. 前端通过 Tauri IPC 调用后端 API
3. Go 后端建立 SSH 连接
4. 终端输出通过 WebSocket 实时推送到前端
5. xterm.js 渲染终端界面

### 文件传输流程

1. 用户在前端发起文件传输操作
2. 前端调用 SFTP API
3. Go 后端创建 SFTP 子系统
4. 文件通过 SFTP 协议传输
5. 传输进度通过 WebSocket 实时更新
6. 全局传输浮标显示所有传输任务状态

### 安全机制

- **认证信息**: 密码通过系统密钥链加密存储
- **主机指纹**: 首次连接时验证主机指纹，后续连接自动验证
- **通信加密**: 所有网络通信使用 SSH 加密协议
- **本地加密**: 敏感配置使用 AES-256-GCM 加密

## 性能优化

- **异步处理**: Go 后端充分利用 goroutine 实现并发
- **连接池**: 复用 SSH 连接，减少重复连接开销
- **流式传输**: 大文件采用流式传输，降低内存占用
- **增量同步**: 文件夹同步仅传输变更的文件
- **断点续传**: 支持传输中断后继续

## 适用场景

### 开发者

- 管理多个开发/测试/生产环境服务器
- 快速部署代码到远程服务器
- 查看远程日志和调试应用
- 同步配置文件到多台服务器

### 运维人员

- 批量管理大规模服务器集群
- 定期同步文件和备份
- 执行自动化运维任务
- 监控多台服务器状态

### 系统管理员

- 安全的远程服务器管理
- 文件权限和用户管理
- 系统配置的批量部署
- 审计和日志管理
